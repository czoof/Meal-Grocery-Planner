# Meal & Grocery Planner

This is a Meal and Grocery Planning web application designed to make weekly meal preparation easier and more efficient. Users can plan meals for the week, save favorite recipes, and automatically generate a grocery list based on their meal plan.

## Core Features
- Weekly meal planning and recipe management
- Automatic grocery list generation
- Check off items while shopping
- Save favorite meals for quick reuse

## Tech Stack
- Frontend: React  
- Backend: Python (Flask)  
- Database: MySQL  
- Deployment: Docker + Google Cloud Run  
- AI Integration (Post-MVP): Automatically suggest meals and adjust ingredients
